To get `GnuPG` working on MacOS:

```shell
$ brew install gpg2 pinentry-mac
```


## GnuPG fatal

```shell
$ git commit -S -m 'Test'
error: gpg failed to sign the data
fatal: failed to write commit object
```

At the end of `~/.zshrc` add:

```
...
export GPG_TTY=$(tty)
```

# This is Git's per-user configuration file.
